ViTSelfAttention and ViTAttention

ViTSelfAttention Class Exploration

Dimentionality Check​

if config.hidden_size % config.num_attention_heads != 0 and not hasattr(config, "embedding_size"):
    raise ValueError(...)

This checks if the hidden size is divisible by the number of attention heads. It ensures that the model's dimensions align correctly for multi-head attention.

Attention Heads Configuration​

self.num_attention_heads = config.num_attention_heads
self.attention_head_size = int(config.hidden_size / config.num_attention_heads)
self.all_head_size = self.num_attention_heads * self.attention_head_size

These lines set up the configuration for the attention heads. It calculates the size of each attention head and the total size of all heads combined.

Linear Layers for Query, Key, Value​

self.query = nn.Linear(config.hidden_size, self.all_head_size, bias=config.qkv_bias)
self.key = nn.Linear(config.hidden_size, self.all_head_size, bias=config.qkv_bias)
self.value = nn.Linear(config.hidden_size, self.all_head_size, bias=config.qkv_bias)

Three linear layers are defined for transforming the input into query, key, and value vectors. These are essential components of the self-attention mechanism.

Dropout Layer​

self.dropout = nn.Dropout(config.attention_probs_dropout_prob)

A dropout layer is added to regularize the attention weights and prevent over-fitting.

transpose_for_scores Method​

def transpose_for_scores(self, x: torch.Tensor) -> torch.Tensor:
    ...

This method reshapes and transposes the input tensor for the multi-head attention operation, preparing it for matrix multiplication.

Forward Method​

def forward(
    self, hidden_states, head_mask: Optional[torch.Tensor] = None, output_attentions: bool = False
) -> Union[Tuple[torch.Tensor, torch.Tensor], Tuple[torch.Tensor]]:

The forward method defines the computation performed at every call. It computes the self-attention using the input hidden_states.

ViTAttention Class Exploration

Constructor​

def __init__(self, config: ViTConfig) -> None:

The constructor takes ViTConfig as an argument, similar to ViTSelfAttention.

Self-Attention and Output Layers​

self.attention = ViTSelfAttention(config)
self.output = ViTSelfOutput(config)

It initializes a ViTSelfAttention instance for computing self-attention and a ViTSelfOutput instance for processing the output of the self-attention layer.

Pruned Heads Initialization​

self.pruned_heads = set()

Initializes an empty set to keep track of any attention heads that are pruned (removed) from the model for efficiency or experimentation.

Prune Heads Method​

def prune_heads(self, heads: Set[int]) -> None:
    ...

This method allows for the removal of specific attention heads, which can be useful for model optimization and interpretability studies.

Forward Method​

def forward(
    self,
    hidden_states: torch.Tensor,
    head_mask: Optional[torch.Tensor] = None,
    output_attentions: bool = False,
) -> Union[Tuple[torch.Tensor, torch.Tensor], Tuple[torch.Tensor]]:

The forward method in ViTAttention calls the forward method of ViTSelfAttention and then processes its output further
